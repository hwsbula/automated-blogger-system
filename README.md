# Automated Blogger System

This repository contains the planning and delivery documents for a multi-client WordPress blogging system.

## Structure

- `01-foundation/`: project brief, assumptions, decisions, and threading notes
- `02-system-design/`: architecture, integrations, technical specification, and implementation plan
- `03-workflows/`: workflow mapping and failure-mode analysis
- `04-operations/`: SOPs, QA checklist, and handoff guidance
- `05-delivery/`: final delivery materials

## Current State

This repository is documentation-first and currently contains system planning, operational guidance, and delivery artifacts.
